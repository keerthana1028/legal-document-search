📊 Information Retrieval Tables

📋 Table 1 — Inverted Index

Shows which documents contain each query term and the document frequency (df).

🔘 Table 2 — Boolean Retrieval Result

Documents retrieved using Boolean AND / OR / NOT on the query.

📈 Table 3 — TF (Term Frequency)

TF(t,d) = count of term t in document d ÷ total tokens in document d

📉 Table 4 — IDF (Inverse Document Frequency)

IDF(t) = log((N+1)/(df+1)) + 1  —  N = total documents, df = docs containing term

⚡ Table 5 — TF-IDF

TF-IDF(t,d) = TF(t,d) × IDF(t) — higher = more important term in that document.

📐 Table 6 — Cosine Similarity

Cosine(Q,D) = (Q·D) / (|Q|×|D|) — angle between query and document TF-IDF vectors.

🏆 Table 7 — Document Ranking

Documents ranked by cosine similarity — Rank 1 is the best match.
